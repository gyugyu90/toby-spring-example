# toby-spring-example

#### Environment
Java 1.8  
Spring Boot 2.1.4  
MySQL 8.0.15


#### SQL
1장
```SQL
CREATE DATABASE TOBYSPRING;

USE TOBYSPRING;

CREATE TABLE USERS (
    id VARCHAR(10) PRIMARY KEY,
    name VARCHAR(20) NOT NULL, 
    password VARCHAR(10) NOT NULL
);

USE TESTDB;

CREATE TABLE USERS (
    id VARCHAR(10) PRIMARY KEY,
    name VARCHAR(20) NOT NULL, 
    password VARCHAR(10) NOT NULL
);
```


5장
```SQL
USE TESTDB;
```
